WsgiDAV Explorer

Help

This is the help page for the WsgiDAV Explorer. It provides a user interface to browse and manage files on a WebDAV server.

Features

  • Browse directories and files on a WebDAV server.
  • Upload files to the server.
  • Create new folders.
  • View file details and metadata.
  • Search for files and folders.
  • Logout from the current user session.
